# Compact Context Workflow

When context is growing but you still want to continue in the same conversation.

## Steps

1. Review all previous exchanges in this conversation.

2. Identify completed tasks vs. active/pending tasks.

3. Create a compressed summary that preserves:
   - All conclusions and decisions (MUST keep)
   - Current file states and modifications (MUST keep)
   - Active task progress (MUST keep)
   - Remove outdated exploratory discussion
   - Remove superseded approaches
   - Remove redundant file readings

4. Present the compact summary to the user and continue from that point.

5. Inform the user: "I've compacted the conversation context. We now have more room to work. Current state: [brief summary of active task]."
